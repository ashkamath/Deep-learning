Inline Question 1:¶

It is possible that once in a while a dimension in the gradcheck will not match exactly. What could such a discrepancy be caused by? Is it a reason for concern? What is a simple example in one dimension where a gradient check could fail? Hint: the SVM loss function is not strictly speaking differentiable

Your Answer: The max(0,x) function involved in the support vector loss function is not differentiable at the point where the values go to negative. This can cause a discrepancy in the gradcheck and is not a cause for concern. Any function, that is differentiable eveywhere except at the point of a kink in the function, can cause this issue. As long as it is piecewise differentiable, it can be used.
